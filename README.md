# Superstore-Sales-Analysis-Dashboard

## Overview
This project focuses on analyzing retail sales data using Python and Power BI to extract meaningful business insights. The dashboard provides a clear view of sales performance, profit trends, category-wise contribution, and regional analysis.

## Objectives
- Clean and preprocess raw sales data
- Perform exploratory data analysis (EDA)
- Identify sales and profit patterns
- Build an interactive Power BI dashboard for decision-making

## Tools & Technologies
- Python
- Pandas
- Google Colab / Jupyter Notebook
- Power BI

## Key Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Sales by Category
- Sales Trend Over Time
- Profit by Region
- Interactive Region Slicer

## Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Exporting Cleaned Dataset
5. Dashboard Creation in Power BI

## Files Included
- `cleaned_superstore.csv` – cleaned dataset
- `Superstore_Sales_Dashboard.pbix` – Power BI dashboard
- `analysis.ipynb` – Python notebook for preprocessing and EDA
- `dashboard_screenshot.png` – dashboard preview

## Outcome
This project demonstrates beginner-level skills in data cleaning, analysis, visualization, and dashboard development using Python and Power BI.
